# reference/ — external source material (read-only)

Isolated reference material from the foundational paper. **Not imported or executed by this
repo's engine, frontend, or tests** — kept here purely for citation and cross-checking.

- `RSPA-2025-0696...pdf` — Yang–You–Rosen 2025, *Proof of folding a tessellated uniform-thick
  plate into compact stacks* (the paper this project builds on).
- `kirigami-programme/` — the authors' original reference implementation (MATLAB Kirigami App +
  supplementary `python files/`, plots, and packaged `.mltbx`/`.zip`). Standalone; this repo does
  not depend on it.
